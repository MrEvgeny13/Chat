# Chat
**Startup instruction:**

1) Launch "Server":

1.1. Port (in console): 4444 (or any other 4 digits);

2) Launch "ClientGuiController":

2.1. Address: localhost || 0.0.0.0 || 127.0.0.1; 

2.2. Port: same (see 1.1);

3) Launch "BotClient":

3.1. Address: same (see 2.1); 

3.2. Port: same (see 1.1);
